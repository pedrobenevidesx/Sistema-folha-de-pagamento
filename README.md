# 💰 Sistema de Folha de Pagamento

Projeto desenvolvido em Python com o objetivo de simular o cálculo de folha de pagamento com base em regras reais, incluindo descontos de INSS, IRPF e vale transporte.

---

## 🚀 Funcionalidades

* Cálculo de INSS de acordo com faixas salariais
* Cálculo de IRPF com base na base de cálculo
* Desconto de vale transporte (6%)
* Cálculo do salário líquido
* Entrada de múltiplos funcionários via terminal

---

## 🛠️ Tecnologias utilizadas

* Python 3
* Lógica de programação
* Organização de funções

## ▶️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/folha-pagamento.git
```

2. Acesse a pasta do projeto:

```bash
cd folha-pagamento
```

3. Execute o arquivo:

```bash
python folhaDePagamento.py
```

## 📌 Regras de cálculo

* INSS: baseado em faixas salariais progressivas
* IRPF: calculado após desconto do INSS
* Vale transporte: 6% do salário bruto
